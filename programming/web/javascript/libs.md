## Read YAML files in Node

<div class="row row-cols-md-2"><div>

#### js-yaml

[js-yaml](https://www.npmjs.com/package/js-yaml) (5.8k ‚≠ê) is frequently updated.

```javascript
const yaml = require('js-yaml');
const fs   = require('fs');

const xxx = yaml.load(fs.readFileSync('test.yaml', 'utf8'));
console.log(xxx);
```
</div><div>

#### yamljs

[yamljs](https://www.npmjs.com/package/yamljs) (0.9k ‚≠ê) was not updated for years.

```javascript
const YAML = require('yamljs');
const fs   = require('fs');

const xxx = YAML.parse(fs.readFileSync('test.yaml', 'utf8'));
console.log(xxx)
```
</div></div>
