# Unordered list

[Go back](..)

Use **ul** and **li** tags to make an unordered list.

```html
<ul>
    <li>one</li>
    <li>two</li>
    <li>...</li>
</ul>

<ul>
    <li>one</li>
    <li>two
        <ol>
            <li>...</li>
            <li>...</li>
        </ol>
    </li>
</ul>
```
