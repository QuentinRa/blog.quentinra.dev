# Ordered list

[Go back](..)

Use ol and **li** tags to make an ordered list.

```html
<ol>
    <li>one</li>
    <li>two</li>
    <li>...</li>
</ol>

<ol>
    <li>one</li>
    <li>two
        <ol>
            <li>...</li>
            <li>...</li>
        </ol>
    </li>
</ol>
```
