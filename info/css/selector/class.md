# class selector

[Go back](..)

Each HTML tag can have a ``class`` attribute,
but you can add a lot of classes by separating
the values with a space.

```html
<p class="text-danger bg-dark">a text here</p>
```

and in your style.css you will use ``.class`` selector

```css
.text-danger {
    color: #dd4441;
}

.bg-dark {
    background: #1c2127;
}
```
