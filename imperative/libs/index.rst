================================
Librairies (C, JAVA, ...)
================================

| :math:`\color{grey}{Version \ 0.2.0}`
| :math:`\color{grey}{Dernière \ édition \ le \ 15/03/2021}`

Ce cours est censé contenir des libraires utiles, graphiques
notamment.

Voici un résumé,

	*
		:code:`NCurses` (ncurses.h/curses.h) : librairie en C native sous linux pour faire des interfaces
		graphiques en ligne de commande, très simplement.
	* :code:`JavaFX` : une librairie graphique en JAVA
	* :code:`lwjgl` : une librairie graphique en JAVA de très bas niveau.
	* :code:`Libgdx` : une librairie graphique en JAVA permettant de faire des Jeux, surcouche à la lwjgl.
	* :code:`litiengine` : une librairie graphique en JAVA, basée sur swing, pour faire des jeux

et vous pouvez trouver les cours complets (qui sont en cours de création) ici
