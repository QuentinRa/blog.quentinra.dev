Portée des variables
○ dans les fonctions → locale
○ hors des fonctions → globale
○ global : pour utiliser une variable globale dans une fonction
○ static : pour conserver la valeur au fil des appels de la fonction
● Variables superglobales
○ $_GET : paramètres d’URL
○ $_POST : valeurs passées par POST
○ $_SERVER : informations sur le script (nom, chemin…), client (IP, port…), serveur…
○ $_SESSION : valeurs stockées dans les sessions
