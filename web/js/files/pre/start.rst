================================
Mon premier script
================================

Vous allez coder du js dans un fichier .js ou dans une balise script.

.. code:: html

	<script>alert('script tourne')</script>
	<!-- ou dans un autre fichier -->
	<script src="file.js"></script>

et file.js

.. code:: js

	// affiche un popup avec 'script tourne'
	alert('script tourne')
