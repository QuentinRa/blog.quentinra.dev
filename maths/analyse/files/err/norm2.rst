==================================
Norme en 2
==================================

:math:`||x||_2 = (\sum_{i=1}^{n}{ |\ x_i |^2} )^{1/2} = \sqrt{\sum_{i=1}^{n}{ |\ x_i |^2}}`

ou :math:`|||A|||_2 = \max_{  y \neq 0 } \frac{||Ay||_2}{||y||_2}`.

Si A est normale alors :math:`|||A|||_2 = \rho (A)` sinon :math:`|||A|||_2 = \sqrt{\rho (A^*A)}`
