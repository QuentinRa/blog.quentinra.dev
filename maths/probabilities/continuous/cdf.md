# Inverse cumulative distribution function

The Inverse cumulative distribution function or Quantile function or `fonction de r√©partition inverse` is the name given when we are trying to find a $k$ given the wanted result of the function $\alpha$.

@
\alpha = \mathbb{P}(X \le k) 
@

It's written $F_X^{-1}(\alpha)$ or $Q_\alpha$.
