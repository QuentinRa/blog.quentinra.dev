Rends avec un atlas

Un texture Atlas est une texture qui contient plusieurs petites textures.

Par exemple un tileSheet contient un ensemble de tile. Les accès disque sont couteux donc il vaudrait mieux les réduire, par exemple avec un atlas.

Tiled

Pour plus de facilités, vous pouvez utiliser Tiled. Vous allez créer vos maps avec une interface graphique.

Il vous faudra coder la toute première fois du code pour afficher une map tiled, sauf si quelqu'un l'a déjà fait dans votre langage.

Faites en sorte que le nombre de niveaux soient la même pour toutes vos maps et qu'ils aient la même sémantique !

Ne pas dessiner les tiles vides (tiled: tile = 0).
