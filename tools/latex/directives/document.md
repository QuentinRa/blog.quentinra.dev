# Document

[Go back](..#writing-documents)

Using a date, a title, and an author, you document
source code might look like this

```latex
\documentclass{article}
% some package
\usepackage[utf8]{inputenc}

\title{Title}
\author{author}
\date{a date}

\begin{document}
\maketitle

some content here
\end{document}
```
