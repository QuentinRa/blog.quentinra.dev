# mystic commands

[Go back](..#-and-now-mastering-it)

You may not use them a lot, but still

* `:!` : execute a command
* `:f nom` : rename file
* `:e fichier` : open another file
* `:r fichier` : print and insert file
* `:!gcc` :  compile inside vim
* `CTRL-G` : show your position inside the file
