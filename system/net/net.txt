Port
	Il s'agit de l'endroit par lequel les données vont transiter.

	Ex : ftp (21), ssh (22), ...

Service
	Les point d'entrées/sorties d'un protocole. Le concept associé est un socket=prise. Chaque
	machine va ouvrir un tuyau à un port, pour faire transiter les données.

PDU (Protocol Data Unit)
	Message échangé. On détermine si c'est int, string, ...

	Pour les entiers l’endianness est spécifiée, le format des flottants, le charset
  pour les caractères et les chaînes de caractères, .
