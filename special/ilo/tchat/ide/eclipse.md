# Eclipse

[Go back](../index.md)

You must follow the [official tutorial](https://openjfx.io/openjfx-docs/#IDE-Eclipse). You may add notes below if needed,

* enter **fx** in the search bar to open a menu to add SceneBuilder path and JavaFX SDK path
* to start a JavaFX application
  * add into your configuration (in **VM** arguments)
  * ``--module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml``
  * with the **correct** ``/path/to/javafx/lib``.
