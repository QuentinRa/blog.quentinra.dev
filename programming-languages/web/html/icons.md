# Icons

<div class="row row-cols-lg-2"><div>

There are many ways to add icons to a website

* [Bootstrap Icons](https://icons.getbootstrap.com/) <small>(6.5k ‚≠ê, üòç)</small>
* [Font Awesome](https://fontawesome.com/v6.0/icons) <small>(70.7k ‚≠ê, attribution required)</small>
* [Feather Icons](https://github.com/feathericons/feather) <small>(23k ‚≠ê)</small>
</div><div>

They all work the same way. In the `head`, add a link to some CSS file.

```html!
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.3/font/bootstrap-icons.css">
```

Then, in the code, use given classes

```html
<!-- https://icons.getbootstrap.com/icons/alarm/ -->
<i class="bi bi-alarm"></i>
```
